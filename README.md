# Anchit's Work — Portfolio

Personal portfolio site for **Anchit Tandon**, Senior Product Manager at Times Internet.

**Live:** [anchits-work.vercel.app](https://anchits-work.vercel.app) *(once deployed)*

## What's here

- Editorial dark/light mode design (champagne accent, Fraunces + JetBrains Mono + Inter)
- Hero, stats strip, selected work (Times Internet), side projects, about, timeline, contact
- Floating AI chatbot trained on a knowledge base about my work and side projects
- Single-file static site — no build step required

## Stack

- Plain HTML / CSS / JS — no framework, no bundler
- Google Fonts (Fraunces, JetBrains Mono, Inter)
- Vercel for hosting
- (Optional) Anthropic Claude API for upgrading the chatbot to real LLM — see `DEPLOY.md`
